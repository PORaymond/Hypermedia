# Le format Json

## Définition

Json
: Le JavaScript Object Notation (JSON) est un format standard utilisé pour représenter des données structurées de façon semblable aux objets Javascript. [(Réf)](https://developer.mozilla.org/fr/docs/Learn/JavaScript/Objects/JSON)

## [Syntaxe](https://www.w3schools.com/js/js_json_syntax.asp)

### Donnée Json
*clé*: *valeur*

#### exemple
```json
"nom": "Miranda"
```

### objet Json
{"clé1": *valeur1*, ..., "clé_n", *valeur_n*}

#### exemple

```json
{
  "nom": "de Gilles",
  "prénom": "Gilles",
  "age": 30
}
```
